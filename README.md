# 🚀 Selenium Automation Learning Project

Este repositorio contiene mis primeros pasos en el mundo del **QA Automation**. Aquí documento el progreso de mi aprendizaje, desde la configuración del entorno hasta la creación de scripts de prueba funcionales.

## 🛠️ Tecnologías y Herramientas
* **Lenguaje:** Java 11 (Adoptium)
* **Gestor de Dependencias:** Maven
* **Framework de Testing:** TestNG 7.10.2
* **Herramienta de Automatización:** Selenium WebDriver 4.25.0
* **IDE:** IntelliJ IDEA

## 🎯 Logros de Hoy
- ✅ Configuración completa del `pom.xml` con dependencias modernas.
- ✅ Creación de un `testng.xml` para la gestión de suites de prueba.
- ✅ Primer test funcional exitoso: Navegación a la web de práctica y validación dinámica de títulos.
- ✅ Implementación de `ChromeOptions` para manejo de sesiones seguras.

## 📁 Estructura del Proyecto
- `src/test/java/tests/`: Clases de prueba (Test Cases).
- `testng.xml`: Configuración de ejecución de pruebas.
- `pom.xml`: Configuración de Maven y librerías.
